Infosys Project - Implementation Summary (November 2025)

1) Admin - User Management
   - File: admin/users/edit.php
   - Added profile photo upload (stores under assets/images/profiles/).
   - Updates user info with mysqli prepared statements; optional password change.

2) Orders - Show Item Thumbnails
   - File: admin/orders/view.php
   - File: user/orders.php/view.php
   - Joined products to fetch image_url and display small thumbnails beside product names.
   - Uses placeholder when image is missing.
   - Updated status badges in admin/orders/index.php and admin/orders/view.php to Bootstrap 5 classes.

3) Discounts - Status Display
   - File: admin/discounts/index.php
   - Shows “Max Usage” badge when usage_limit is set and reached (times_used >= usage_limit).
   - Updated Bootstrap badges to bg-* variants.

4) Reports - Inventory Report Status
   - File: admin/reports/inventory_report.php
   - Corrected Bootstrap badge classes for status display.
   - Access now allowed for Inventory Manager (requireAdminOrInventoryManager()).
   - Shows Inventory Manager navbar when applicable.

5) Inventory Manager Dashboard and Navbar
   - File: admin/inventory_dashboard.php (new)
   - File: includes/inventory_navbar.php (new)
   - Inventory-focused dashboard with stats and recent restocks.
   - Dedicated navbar limited to inventory-related pages.
   - Inventory pages select navbar based on role:
     - admin/inventory/index.php
     - admin/inventory/history.php
     - admin/inventory/restock.php

6) Expenses Report
   - File: admin/reports/expenses_report.php (new)
   - Summarizes total expenses (restocking_transactions) and lists recent restock operations.
   - Linked under Reports in admin navbar.
	- Added optional date range filter using prepared statements (start_date, end_date).

7) Net Income Report with Date Filter
   - File: admin/reports/net_income_report.php (new feature)
   - Calculates Total Revenue (non-cancelled orders), Total Expenses (restocking), and Net Income.
   - Added optional start/end date filter using prepared statements.
   - Linked under Reports in admin navbar.

8) Admin Navbar Role Awareness
   - File: includes/admin_navbar.php
   - Detects role and hides admin-only menus from Inventory Manager.
   - Adds links for Expenses and Net Income reports (Admin only).

9) Inventory History Styling and Access
   - File: admin/inventory/history.php
   - Access extended to Inventory Managers and badge classes aligned with Bootstrap 5.

10) Password Reset Verification Code Flow
   - File: user/auth/forgot_password.php
   - File: user/auth/reset_password.php
   - Sends a 6-digit verification code via email using PHPMailer and stores it in session with expiry.
   - Reset page validates the code before allowing password update (new password + confirmation).

Notes
 - All SQL writes/updates use prepared statements with bind_param.
 - Only include()/require() used; no require_once or __DIR__.
 - Badge classes migrated to Bootstrap 5 (bg-*).
 - No linter issues detected in modified files.
